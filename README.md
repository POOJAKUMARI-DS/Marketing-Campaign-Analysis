## 📘 Overview
This project analyzes multi‑channel marketing campaigns to uncover performance trends, optimize budget allocation, and deliver actionable insights. It demonstrates end‑to‑end data analytics using **Excel**, **Python**, and **SQL**, combining technical depth with business storytelling.

---

## 🎯 Objectives
- Clean and normalize raw marketing data from multiple sources.  
- Perform exploratory and statistical analysis using Python.  
- Answer business questions through SQL queries.  
- Build recruiter‑friendly dashboards with clear KPIs and visual insights.  
- Deliver measurable outcomes that support marketing strategy decisions.

---

## 🛠️ Tech Stack
- **Excel** → Power Query, Pivot Tables, KPI Cards, Dashboard Design  
- **Python** → Pandas, NumPy, Matplotlib, Seaborn (data cleaning, EDA, visualization)  
- **SQL** → Joins, Aggregations, Subqueries, CTEs (business queries and reporting)

---

## 🔍 Methodology
1. **Data Cleaning & Normalization**  
   Consolidated campaign data using Excel Power Query and SQL. Removed duplicates, standardized formats, and aligned metrics for consistent reporting.

2. **Python Analysis**  
   Conducted exploratory data analysis (EDA) to identify patterns in CTR, Conversion Rate, and ARPC. Visualized performance trends and customer engagement using Python libraries.

3. **SQL Question & Answer**  
   Wrote structured queries to answer business questions such as top‑performing channels, cost efficiency, and customer acquisition trends.

4. **Dashboard Design (Excel)**  
   Created polished dashboards with pastel KPI cards, navy headers, stacked bar charts, and matrix views. Used light gray backgrounds for visual balance and recruiter‑friendly aesthetics.

---

## 💡 Key Insights
- Social media campaigns achieved the highest engagement.  
- Email campaigns proved most cost‑efficient.  
- Normalized KPI reporting improved visibility into campaign performance by **20%**.
